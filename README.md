# Desktop Grade Converter App
 Desktop Grade Converter App

The **Desktop Grade Converter App** converts numeric scores into letter grades. It is easy to use and helps students and teachers quickly see grades and pass/fail status.
BY SHOWING GRADES 
| Score (%) | Grade | Remarks   |
| --------- | ----- | --------- |
| 90–100    | A     | Excellent |
| 80–89     | B     | Very Good |
| 70–79     | C     | Good      |
| 60–69     | D     | Pass      |
| 0–59      | F     | Fail      |
