# desktop-grade-converterThis project is a Student Grade Converter desktop application built with Kotlin and Jetpack Compose for Desktop. It allows users to browse and select an Excel file containing student names and marks, and automatically converts the numeric marks into grades using a standard grading scale (A: 80–100, B: 70–79, C: 60–69, D: 50–59, F: below 50). The converted grades are displayed immediately in the application’s interface for quick review.

The application uses Apache POI to read Excel files, extract student marks, and convert them into grades programmatically. The app’s structure separates UI code, data models, and Excel reading/exporting logic to make the project organized and maintainable. Users can see each student’s grade in the app and can optionally export the results to a new Excel file for record keeping.

This project is suitable for teachers, students, or anyone who wants to quickly calculate grades from a set of marks without manual processing. It demonstrates the use of modern Kotlin desktop development with Jetpack Compose and integration with Excel files, providing a simple and interactive way to manage student grades efficiently.
